Team Fortress 2 map : cp_neighbor

For practice purposes, I am working on a Team Fortress 2 custom map using the Hammer editor.
Team Fortress 2 is a 2007 team-based first person shooter by Valve.
In this game mode, BLU team has to capture point A and then B, while RED has to defend both.

About:

Most TF2 maps are set in rural landscapes and don't feature any streets or tall buildings. My personal goal with this project was to create a map with an interesting layout despite being based on a rural neighborhood, which usually has plenty of empty straight lines.
To achieve this, most of the fights do not actually happen in the streets, and I am taking inspiration on cities built on mountains such as San Francisco and Montreal to justify streets having varying heights and odd turns.

My tasks:

Level design



Control point A

The first control point is meant to be easier for BLU to take. It is set next to a shipping facility with lots of paths to it, making BLU ambushes possible from all directions. However, RED has multiple places for a viable sentry nest and can lock down some of those paths.


> <


Control point B

The second control point is set in the courtyard of a 1960s style hotel. It is much more cramped and is easier for RED to defend. It is closer to the RED spawn room, and BLU has no path dedicated to taking out the good sentry spot. BLU will need to focus their efforts on aggresive pushes rather than ambushing from all directions.


> <


Initial sketch

This the plan I was satisfied with after my first sketches. I started blocking using this as a reference for scale.
After testing, the map became more narrow, the building on the left of Control Point A became bigger and gave RED a good spot for a sentry, and some line of sights got shortened to make it harder for Snipers.
